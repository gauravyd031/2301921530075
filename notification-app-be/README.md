## Notification App Backend
The Notification App Backend is a Node.js and Express.js application that fetches notifications from the evaluation service using authenticated API requests. It integrates the reusable logging middleware to log all backend operations and returns notification data through REST APIs.

### Features

- Fetch notifications from the evaluation service.

- Secure API communication using Bearer Token authentication.

- Reusable logging middleware integration.

- Structured Express.js project architecture.

- Error handling for failed API requests.

### Tech Stack

- Node.js

- Express.js

- Axios

- Dotenv
