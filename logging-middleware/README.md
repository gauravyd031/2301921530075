## Logging Middleware
The Logging Middleware is a reusable Node.js module that records backend events by sending logs to the evaluation logging service. It provides a centralized logging function that can be used across multiple backend applications to track application flow, API requests, and errors.

### Features

- Reusable logging module.

- Sends logs to the evaluation logging service.

- Supports different log levels (info, warn, error, etc.).

- Validates log inputs before sending requests.

- Easy integration with multiple backend projects.

### Tech Stack

- Node.js

- Axios

- Dotenv
